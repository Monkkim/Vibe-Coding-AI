전체 구조 개요
수강생들의 학습 여정을 체계적으로 관리하고, 게임화 요소를 통해 동기부여하며,
성장의 핵심 지점을 발견하도록 돕는 통합 플랫폼입니다.
1. 크랙 타임 (CRACK TIME)
핵심 개념
크랙(CRACK)은 세 가지 의미를 담고 있습니다:
통찰의 균열 (Insight Crack): 단단한 사고방식에 금을 내어 핵심 인사이트를 발견
성장으로의 돌파구 (Breakthrough Point): 최소 노력으로 최대 효과를 내는 결정적 지점
새로운 시작의 순간 (Moment of Clarity): 명료한 상태에서 새로운 원칙을 세우는 전환점
기능 요구사항
일일 대표님 리포트 생성
날짜별로 자동 생성 (예: 2025.12.18)
각 수강생별 개별 리포트 페이지
현재의 안개 섹션
현재 직면한 문제나 상황을 한 줄로 정리
고민 상황을 명료하게 표현하는 입력 필드
관점의 빛 섹션 (크랙 포인트)
문제를 바라보는 새로운 관점이나 핵심 통찰 기록
강조 표시 및 하이라이트 기능
예시: "올들라는 마음에 '세계'라는 빛을 내린 시간!"
오늘의 지도 (Next Action)
구체적인 행동 계획을 체크리스트로 작성
완료 여부 체크 기능
다음 단계로의 연결성 표시
디자인 요소
상단에 다이아몬드 아이콘과 "CRACK TIME" 로고
따뜻하고 밝은 색상 (오렌지, 노란색 포인트)
카드 형식의 섹션 구분
하단에 "MORNING SUNLIGHT · BREAKING LIMITS · CRACK TIME" 슬로건
2. 가치 토큰 게임 (Value Token GAME)
핵심 개념
수강생들이 서로의 가치를 인정하고 교환하는 게임화된 동기부여 시스템
기능 요구사항
기수별 게임 보드
각 기수별로 독립적인 게임 보드 운영 (예: AG 42기 가치 토큰 GAME)
기수별 전체 생태계 총 유동량 표시 (예: 230만원)
목표 달성률 표시 (예: 10,000만원 달성까지 97,700,000원 남음)
개인 프로필 & 레벨 시스템
캐릭터 아바타 (새싹 아이콘에서 성장)
레벨 표시 (Lv.2 숙련가 등)
현재 보유 토큰/목표 토큰 진행바
오늘 목표/주간 목표 표시 (예: 0만/10만, 0만/50만)
도착할 토크라인 표시 (다음 레벨까지 필요 금액)
가치 인정 토큰 발행
가치 선택 드롭다운: 어떤 가치를 인정하는지 선택
금액 설정: 3만원, 5만원, 7만원, 10만원 버튼 (사용자 정의 가능)
직접 입력: 최대 10명까지 수신자 선택
메시지 작성: 인정하는 가치에 대한 구체적 메시지 입력
토큰 발행하기 버튼으로 전송
명예의 전당
이번 주 순위 표시 (금, 은, 동, 노력상)
각 수상자가 받은 가치 설명 (예: "토끼 및 팀포 케어 + 룰렛에")
실시간 활동 피드
누가 누구에게 얼마를 보냈는지 실시간 표시
메시지 내용 미리보기
시간 기록 (예: 12.19 10:11)
활동 로그 타임라인
상단 메뉴 탭
가이드

실시간 피드 강조

제작해 드린 Vibe Coding Consulting 통합 대시보드는 요청하신 네 가지 핵심 모듈을 고도화된 UI/UX와 함께 구현했습니다.
주요 업데이트 사항:
크랙 타임 (CRACK TIME):
따뜻한 오렌지/옐로우 톤의 전용 테마를 적용했습니다.
'현재의 안개', '관점의 빛', '오늘의 지도' 섹션을 명확히 구분하여 매일의 통찰을 기록할 수 있습니다.
다이아몬드 로고와 슬로건을 배치하여 아이덴티티를 강화했습니다.
가치 토큰 게임 (Token Game):
게임 보드 형식으로 생태계 유동량과 목표 달성률을 시각화했습니다.
레벨 시스템과 성장형 아바타를 포함한 개인 프로필 카드를 추가했습니다.
3, 5, 7, 10만 원 단위의 간편 발행 시스템과 실시간 활동 피드(Activity Feed)를 구현했습니다.
기수별 관리 시스템 (Batch Manager):
폴더 트리 구조를 통해 기수와 수강생을 체계적으로 분류했습니다.
수강생 선택 시 노션(Notion) 스타일의 개인 저널링 보드가 나타나도록 설계했습니다.
Notion API 연동을 위한 바로가기 기능과 페이지 동기화 버튼을 배치했습니다.
세일즈 머신 (Sales Machine):
신규 유입부터 최종 등록까지의 파이프라인 단계를 한눈에 볼 수 있도록 시각화했습니다.
리드별 전환 확률과 히스토리를 관리할 수 있는 테이블과 매출 트렌드 차트를 포함했습니다.
디자인 컨셉:
전체적으로 '아이보리/크림(Ivory/Cream)' 톤의 배경과 부드러운 그림자를 사용하여 고급스럽고 차분한 분위기를 연출했습니다. 각 기능마다 고유한 포인트 컬러(앰버, 에메랄드, 블루, 로즈)를 사용하여 직관적인 네비게이션을 돕습니다.
이제 수강생들은 이 대시보드를 통해 자신의 성장을 시각적으로 확인하고, 서로의 가치를 인정하며 더 큰 도약을 꿈꿀 수 있습니다.

크랙타임의 경우에는 이제 고민을 적으면 그걸 기반으로 해서 해결법을 알려주는거고 가치토큰은 굿 건들 필요가 없음
기수 관리 부분에서는 폴더를 생성하고 삭제하는 기능이 있었으면 좋게고 전체적으로 웹사이트가 실질적으로 움직였으면 좋겠어
버튼을 클릭하면 동작이 된다던지
그리고 각 이름을 클릭하면 오늘의 성찰이 아니고 아침 저널링으로 바꿔주고
내가 적은, 기존의 것을 클릭하면 노션처럼 오른쪽에 보드가 나와줬으면 좋겠어


노션 연동 빼고 저널링 작성클릭하면 오른쪽에서 노션퍼럼 보드가 나오라고
그리고 대시 보드 라이트모드 , 다크모드 선택할 수 있게 해주고
전체 대시보드 크기가 너무 작어 중앙부분 더 키워줘
그리고 로그인 기능은 일단 빼
가치토큰의 경우에는
기수관리에 있는 사람에게 보낼 수 있는거임

입장하기 버튼이 포함된 고품격 랜딩 페이지를 추가하고, 사용자의 요청대로 기수 관리 및 대시보드 구조를 대폭 개선했습니다.
주요 업데이트 사항:

전체 공간 확대: 중앙 콘텐츠 영역의 최대 너비를 1800px로 확장하여 시원한 작업 환경을 제공합니다.
가치 토큰:
기수 관리에 등록된 수강생 리스트를 기반으로 토큰을 발행할 수 있도록 연동했습니다.
다크 모드 & 라이트 모드:
사이드바 하단에서 테마를 자유롭게 전환할 수 있으며, 모든 컴포넌트가 다크 모드에 완벽히 대응합니다.
